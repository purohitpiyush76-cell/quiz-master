# Quiz Master

A simple and interactive Quiz Master application built with **React**, **Vite**, and **React Hooks**. This project demonstrates how to manage application state using the `useState` hook while creating a dynamic multiple-choice quiz.

Users can answer questions one at a time, track their score, and restart the quiz after completion.

---

##  Features

*  Displays one question at a time.
*  Multiple-choice answer options.
*  Real-time score tracking.
*  Automatically moves to the next question.
*  Displays the final score at the end of the quiz.
*  Restart Quiz functionality.
*  Fast development experience with Vite.

##  Getting Started

### Clone the repository

```bash
git clone https://github.com/purohitpiyush76-cell/quiz-master.git
```

### Navigate to the project folder

```bash
cd quiz-master
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```

Open the local URL displayed in the terminal (usually `http://localhost:5173`) to view the application.
